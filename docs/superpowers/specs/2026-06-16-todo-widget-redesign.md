# 待办插件重设计规格

## 背景

当前 widget-todo 是一个基础的增删改查列表，缺乏信息层次和组织能力。本次重设计目标：卡片轻量展示 + 快速勾选，完整交互放在 expand 弹窗里，整体风格对齐飞书任务管理的操感。

## 设计方向

**方向 B · 分组摘要**：

- 卡片：分组折叠/展开 + 优先级彩色圆点 + 截止日期，高度自然截断
- Expand 弹窗：完整的任务管理界面

## 卡片（card view）

### 布局

```
[ 未完成  3 ]  [ 全部]                    [ 展开 ↗ ]
─────────────────────────────────────────────────────
▾ 本周                                            2
  ● [✓] 补齐 widget 尺寸菜单              12/31
  ● [✓] 清理插件设置项
▾ 下周                                            1
  ● [✓] 更新组件文档                       1/8
```

### 交互

- 顶部 tab 切换 未完成 / 全部
- 点击分组标题折叠/展开
- 点击勾选框直接完成
- 点击"展开 ↗"按钮调用 `props.host.openExpand()` 打开弹窗
- 超出高度自然截断（CSS overflow），不加"+ N 项"计数

### 字段显示

- 优先级彩色圆点（高:红 / 中:橙 / 低:蓝 / 无:灰）
- 截止日期（已过期显示红色）
- 不显示：负责人、创建人、描述（留给弹窗）

## Expand 弹窗（expand view）

### 布局

```
待办                                      [ + 新建任务 ]
─────────────────────────────────────────────────────
[ 列表 ]  [ 看板 ]
─────────────────────────────────────────────────────
[ 未完成 ] [ 全部]               [ 排序 ] [ 分组 ] [ 字段 ]
─────────────────────────────────────────────────────
     任务标题              优先级    截止时间    负责人
─────────────────────────────────────────────────────
▾ 本周                                             2
  [✓] 补齐 widget 尺寸菜单    ● 高    12/31    毕金风
  [✓] 清理插件设置项          ● 中    —        毕金风
  + 新建任务
▾ 下周                                             2
  [✓] 更新组件文档            ● 低    1/8      蔡文静
  [✓] 复核 Dashboard 布局     ● 高    —        毕金风  (已完成)
  + 新建任务
+ 新建分组
```

### 功能列表

**视图切换**

- 列表视图（默认）
- 看板视图（按分组显示列，本期留空占位即可）

**过滤**

- 未完成 / 全部 两个 tab

**工具栏按钮**（本期只做 UI 占位，不实现逻辑）

- 排序、分组、字段

**分组**

- 可折叠/展开
- 每组底部有"+ 新建任务"按钮
- 底部有"+ 新建分组"按钮

**任务行**

- 勾选完成（立即生效）
- 点击行展开任务详情（本期暂不实现侧边抽屉，留 TODO）
- 优先级 tag（高/中/低/无）
- 截止日期（过期红色）
- 负责人头像 + 姓名（本期硬编码，无用户系统）

**新建任务**

- inline 输入框，回车确认，Esc 取消
- 默认归属当前分组

**新建分组**

- inline 输入框，回车确认，Esc 取消

## 数据结构

```typescript
type Priority = "high" | "medium" | "low" | "none"

type TodoItem = {
  id: string
  text: string
  done: boolean
  priority: Priority
  dueDate?: string // ISO date string
  groupId: string
  assignee?: string // 本期暂留
}

type TodoGroup = {
  id: string
  name: string
  collapsed: boolean
}
```

持久化键名：`v2_items`、`v2_groups`（与当前已实现版本兼容）

## 验收标准

1. 卡片：显示分组、优先级圆点、截止日期；可勾选；点"展开"能打开弹窗
2. 弹窗：未完成/全部过滤生效；分组可折叠；可新建任务和分组；优先级、截止日期可见
3. 数据持久化：刷新后数据不丢失
4. `pnpm test` 和 `pnpm check` 通过
